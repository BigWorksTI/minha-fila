# Roadmap (Sprint 1)

1) API Laravel básica (auth mínima + CRUD de pedidos)
2) Broadcasting funcionando via Soketi (evento OrderUpdated)
3) PWA Next.js consumindo pedidos em tempo real
4) Admin básico para criar e mudar status
5) Painel TV inicial (listas: preparando, prontos)
6) Deploy backend na VPS via Docker Compose
7) Deploy frontend na Vercel

Próximas iterações (idéias)
- Relatórios (tempo médio, volume/hora)
- Multiloja / multiusuários
- Auditoria e métricas no backend
